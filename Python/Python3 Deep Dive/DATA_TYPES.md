# Python Data Type Hierarchy

### Numbers
- Integral
  - Integers
  - Booleans
- Non-Integral
  - Floats (c doubles)
  - Complex
  - Decimals
  - Fractions! 1/3 + 1/3 + 1/3 = 1

### Collections
- Sequences
  - Mutable
    - Lists
  - Immutable
    - Tuples
    - Strings
- Sets
  - Mutable
    - Sets
  - Immutable
    - Frozen sets
- Mappings
  - Dictionaries

### Callables
- User-defined functions
- Generators
- Classes
- Instance methods
- Class instances (\_\_cal__())
- Builtin functions
- Builtin methods

### Singletons
- None
- Notimplemented
- Ellipsis (...)
